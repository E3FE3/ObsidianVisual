# История изменений (Changelog)

## [1.1.0] - Polish Update (Refinement)
### Улучшено
- **Боковая панель**: Активный элемент теперь имеет сплошную "огненную" заливку (#E85D3D) и белый текст, точно повторяя стиль выделения в боковой панели Apple-приложений.
- **Таблицы**: Переработан стиль таблиц. Удалены вертикальные границы и фоны карточек. Теперь они выглядят как чистые списки (List View) в Apple Music.
- **Чистота интерфейса**: Блок метаданных (Properties/Frontmatter) теперь скрыт по умолчанию для создания ощущения нативного приложения.
- **Документация**: Обновлено руководство пользователя с критически важными инструкциями по настройке "Window frame style: Hidden".

## [1.0.0] - Initial Release
### Добавлено
- Новая тема размещена в `.obsidian/themes/AppleMusic/`.
- Реализован акцентный цвет "Огненный оранжевый" (`#E85D3D`) по всему интерфейсу.
- Боковая панель в стиле Apple Music.
- Современный стек шрифтов (`SF Pro`, `Inter`).

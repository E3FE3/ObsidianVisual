# LAUNCHER - Хранилище команд инфраструктуры

Централизованный репозиторий команд и скриптов для управления инфраструктурой с профессиональным dashboard в стиле **Apple Music Desktop**.

## Дизайн система

### Стиль: Apple Music Desktop (2025)
- **Тема**: Полноценная тема Obsidian **"AppleMusic"** (Dark Mode).
- **Акцент**: Огненный оранжевый (**#E85D3D**) - теплый и энергичный.
- **Интерфейс**:
    - Скрытая левая лента (Ribbon) для минимализма.
    - Полупрозрачная боковая панель с эффектом размытия (Blur).
    - Скругленные углы и мягкие тени.
    - Нативная типографика Apple (SF Pro / Inter).

### Архитектура
Теперь проект использует единую тему `.obsidian/themes/AppleMusic/theme.css` вместо набора разрозненных сниппетов. Это обеспечивает лучшую производительность и цельность дизайна.

## Быстрый старт

### 1. Активация темы
Тема должна активироваться автоматически. Если этого не произошло:
1.  Откройте **Settings** → **Appearance**.
2.  В разделе **Themes** выберите **AppleMusic**.
3.  Установите **Base color scheme** в **Dark**.

### 2. Важно для Windows
Чтобы убрать стандартную белую рамку окна и получить настоящий "App-like" вид:
1.  Перейдите в **Settings** → **Appearance**.
2.  Установите **Window frame style** в **Hidden**.
3.  Перезагрузите Obsidian.

### 3. Рекомендуемые плагины
Для полного погружения (скрытие скроллбаров, заголовков):
-   Установите плагин **Hider**.
-   См. файл `RECOMMENDED_PLUGINS.md` для деталей.

## Особенности дизайна

### Интерфейс
-   **Боковая панель**: Навигация в виде "таблеток" (Pills) с оранжевой подсветкой активного элемента.
-   **Таблицы**: Чистый дизайн без вертикальных границ, прозрачный фон, оранжевые ссылки.
-   **Шрифты**: Используются системные шрифты (`-apple-system` на Mac, `Segoe UI` на Windows), что делает приложение родным для ОС.

### Навигация
-   **Home**: Стартовая страница с карточками.
-   **All Commands**: Полный реестр через Dataview.

## Добавление команд

### Используя шаблон
1.  Создайте заметку, вставьте шаблон `[[_Templates/Command]]`.
2.  Заполните YAML-метаданные (система, технология, теги).

## Устранение проблем

### Интерфейс выглядит "стандартным"
Скорее всего, тема не применилась из-за конфликта со старыми сниппетами.
1.  Зайдите в **Settings** → **Appearance** → **CSS snippets**.
2.  Отключите `00-reset`, `01-variables`, `06-layout`.
3.  Или просто используйте файл `USER_GUIDE.md` для справки.

### Иконки папок
Используйте плагин **Iconize** для установки иконок на папки в боковой панели, чтобы они соответствовали стилю (как на скриншотах).

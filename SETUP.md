# :LiPalette: Настройка LAUNCHER Dashboard

## :LiZap: Быстрая настройка (Новый метод)

Вместо сложной настройки множества файлов, теперь всё работает "из коробки" благодаря теме **AppleMusic**.

### Шаг 1: Проверка темы
1. Откройте **Settings** (`Ctrl+,`) → **Appearance**.
2. Убедитесь, что в поле **Themes** выбрано **AppleMusic**.
3. Если нет — выберите её из списка.

### Шаг 2: Настройка окна (Windows)
1. В том же разделе **Appearance**.
2. Найдите **Window frame style**.
3. Установите **Hidden**.
4. Перезагрузите Obsidian (`Ctrl+R`).

### Шаг 3: Плагины
Для корректной работы Dashboard и автоматизации:
1. **Dataview** (Обязательно) — для таблиц и списков.
2. **Hider** (Рекомендуется) — для скрытия лишних элементов интерфейса (см. `RECOMMENDED_PLUGINS.md`).

### Шаг 4: Готово
Откройте `_Dashboard/Home.md`.

## :LiSparkles: Особенности темы AppleMusic

### :LiTarget: Визуальный стиль
- **Огненный акцент**: Все активные элементы подсвечены цветом `#E85D3D`.
- **Чистота**: Убраны лишние границы, отступы увеличены для легкости чтения.
- **Фокус**: Левая лента (Ribbon) скрыта, чтобы ничто не отвлекало от контента и навигации.

### :LiBug: Решение проблем

#### Тема выглядит "сломанной"
Вероятно, включены старые CSS-сниппеты, которые конфликтуют с темой.
1. Перейдите в **Settings** → **Appearance** → **CSS snippets**.
2. Отключите всё, что начинается с `00-`, `01-`, `02-`.
3. Оставьте только функциональные сниппеты (например, для Callouts или Grid), если они вам нужны.

#### Таблицы не красивые
Убедитесь, что сниппет `02-tables` **ОТКЛЮЧЕН**. Тема AppleMusic имеет встроенные, более современные стили для таблиц.

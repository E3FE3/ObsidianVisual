# rclone

Инструмент для монтирования облачного и удалённого хранилища.

## Описание
rclone - программа командной строки для управления файлами в облачном хранилище и монтирования удалённых файловых систем как локальных дисков. Используется для монтирования VPS серверов как локальных директорий.

## Официальная документация
- [rclone.org](https://rclone.org/)
- [rclone mount документация](https://rclone.org/commands/rclone_mount/)

## Частые сценарии использования
- Монтирование VPS как локального диска в Windows
- Синхронизация файлов между системами
- Резервное копирование в удалённое хранилище
- Просмотр удалённых файлов локально
